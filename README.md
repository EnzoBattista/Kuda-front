# 💻 Gimnasio Kuda - Frontend

Guia corta y al pie para levantar el front.

## 📦 Instalacion

En esta carpeta (`GimnasioKuda-frontend-main`), corre:

```bash
npm install
```

## 🚀 Arranque

Para prender el frontend:

```bash
npm start
```

Tambien podes usar:

```bash
ng serve
```

## 🌐 Como entrar

Abri el navegador y entra a:

**`http://localhost:4200`**
